# Student-Record-Management-System

## Project Overview

A full-stack CRUD application for managing student records using React, Node.js, Express, and MySQL.

---

## Features

- Add Student
- Update Student
- Delete Student
- Search Student
- View All Students
- Form Validation
- Responsive UI

## Tech Stack

Frontend

- React + Vite
- Axios
- React Router 

Backend

- Node.js
- Express.js

Database
- MySQL

Tools
- Postman

---

## Folder Structure

client/
server/

---

## Installation

### Backend

cd server

npm install

node server.js


### Frontend

cd client

npm install

npm run dev
